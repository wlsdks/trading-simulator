## 핵심 포인트 (요약)

- **포지셔닝:** 교육형 ∩ 게임형의 빈 사분면. "돈 잃지 않고 좋은 투자 습관을 익히는 재미있는 연습장." 게임화로 리텐션을 만들되 보상은 **학습·리스크관리**에 건다(도박화 방지 = 규제 방어선 = 차별화).
- **MVP 선결 과제:** 현재 코드의 회계 결함(실현손익 미집계, 더블탭 현금 음수 레이스, 수익률 분모 비교불가)을 게임화·리더보드보다 먼저 해결.
- **데이터 전략:** 초기엔 시뮬레이션 유지, 실시간은 라이선스·비용 검토 후 단계 도입(크립토 거래소 API는 저렴).
- **수익화:** 구독·B2B(학교 라이선스) 중심, 광고는 가드레일 하에 보조.

## 생성 방식

이 문서들은 30여 개의 전문가 페르소나 서브에이전트(PM·UX·트레이딩 도메인·RN 아키텍트·데이터·게임화·교육·그로스·법무·QA + 4개 렌즈 비평단)를 병렬로 가동해 작성되었습니다.
