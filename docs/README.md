### 포지셔닝 (2026-07 확정)
**"재미로 하는 주식 게임"** — 교육 앱이 아니라 가짜 돈으로 즐기는 **재미·경쟁·소셜·수집** 중심의 트레이딩 게임. (초기 기획의 "교육형" 프레임은 폐기.) 법적 가드레일 최소 4개만 유지: ①실제 현금 상금 X ②"모의 시뮬레이션" 면책 ③개인 맞춤 매수추천 X ④실시간 실데이터시 라이선스.

### 감사 이력 (Round 1~3 + 재미 중심 재정렬)
정책·규칙 깊이는 출시급으로 다졌고(12도메인·152정책), 3라운드 감사로 정합성·아티팩트를 정리한 뒤 **재미 중심으로 전면 재정렬**(교육·페다고지·학습성과 계측 드롭, 웰빙 경량화)했습니다. 감사 상세: [planning-review.md](./planning-review.md) · [planning-review-r3.md](./planning-review-r3.md).

## 핵심 포인트 (요약)

- **포지셔닝:** 재미·경쟁·소셜·수집 중심의 **주식 게임**(가짜 돈). 게임화(레벨·배지·대회·리더보드·시즌)로 리텐션을 만든다.
- **MVP 선결 과제:** 현재 코드의 회계 결함(실현손익 미집계, 더블탭 현금 음수 레이스, 수익률 분모 비교불가)을 게임화·리더보드보다 먼저 해결.
- **데이터 전략:** 초기엔 시뮬레이션 유지, 실시간은 라이선스·비용 검토 후 단계 도입(크립토 거래소 API는 저렴).
- **수익화:** 구독·광고(가드레일 하)·인앱결제 중심, B2B/커뮤니티 라이선스는 옵션. 재무 단일소스는 [monetization-model.md](./monetization-model.md).
